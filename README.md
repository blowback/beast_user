# Beast User 

## Creating new blog posts

1. Create new post in `_posts` with title `YYYY-MM-DD-title.md` (where `title` can contain hyphens)
2. Add front-matter: 
``` 
title: blah blah
date: YYYY-MM-DD HH:MM:SS +/-TTTT # or leave blank for unpublished 
categories: [ TOP_CATEGORY, SUB_CATEGORY ] # max of 2! 
tags: [TAG] # lowercase!
```
